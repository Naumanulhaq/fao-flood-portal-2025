# Geospatial Flood Impact Assessment Portal | Pakistan 2025
### Food and Agriculture Organization of the United Nations (FAO)

This portal provides an interactive visualization of the rapid geospatial assessment conducted by the FAO Geospatial Unit and the Pakistan Country Office. It quantifies the impacts of the 2025 monsoon floods on agricultural land, major crop types, and population exposure across Khyber Pakhtunkhwa (KP), Punjab, and Sindh.

## 📌 Assessment Overview
During August and September 2025, intense monsoon rainfall triggered widespread inundation. This project serves as a decision-support tool for emergency response and early recovery planning by delivering spatially explicit information at both provincial and district levels.

### Key Indicators Analysed:
- **Flood Extent**: Analysis of peak inundation periods using Sentinel-1 multi-temporal imagery.
- **Agricultural Impact**: Overlay of flood masks with newly developed 2023 crop-type maps.
- **Population Exposure**: Estimation of human impact using WorldPop (2020) population grids.
- **Recovery Monitoring**: Comparative analysis between peak flooding and October 2025 recession.

## 🛠️ Framework & Methodology
The assessment leverages state-of-the-art geospatial technologies:
- **Satellite Data**: Sentinel-1 SAR (10 m resolution) for robust flood detection regardless of cloud cover.
- **Processing**: Fully implemented in **Google Earth Engine (GEE)** for rapid and reproducible analysis.
- **Visualization Stack**: This portal is built with Vanilla HTML5/CSS3 and integrated with **Chart.js** for dynamic data representation.
