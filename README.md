# Flood 2025 Web Portal

A professional, data-driven web portal for institutional flood statistics.

## Project Goal
This project provides a visually engaging and informative dashboard for visualizing flood-related data for 2025.

## Deployment
This project is designed to be hosted on **GitHub Pages**.

## Features
- **Dynamic Visuals**: interactive charts and counters.
- **Premium Design**: clean, executive-style UI with responsive components.
- **Data Integration**: Scalable structures for statistics.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- ES6 Javascript
- Chart.js for data visualization
- Google Fonts (Inter)
