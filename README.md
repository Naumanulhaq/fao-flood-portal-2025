# Flood 2025 Web Portal | UNFAO Pakistan

A professional, data-driven web portal for visualizing flood-related statistics across KP, Punjab, and Sindh for the year 2025.

## 📊 Project Scope
This dashboard provides interactive visualizations for:
- National flood extent and agricultural impact.
- Provincial comparisons (KP, Punjab, Sindh).
- Crop-type vulnerability (Rice, Cotton, Sugarcane, Maize).
- District-level rankings by impact.
- Recovery monitoring (October 2025).

## 📄 Official Publication & Citation

## 🚀 Deployment
This project is optimized for **GitHub Pages**.

### How to use this repository:
1. Clone the repository.
2. Open `index.html` in a browser to view the dashboard locally.
3. To host, enable GitHub Pages in Settings > Pages using the `main` branch.
